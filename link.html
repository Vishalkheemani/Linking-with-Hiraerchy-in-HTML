Link:

  1. Absolute Link:

    It allows visiting from one website to another.

    Example:
    Google
    Yahoo
  2. Relative Link:

    It allows visiting multiple pages within the same website.

    Example:
    Home Page
    List Page
    Nested Page
    Netlify Page
  3. On Page Link:

    It allows visiting from one part of the page to another part/section.

To learn more about links, visit w3schools.
